# Galaxia Clone

A space shooter game inspired by the classic Galaxia. Built with HTML5 Canvas and vanilla JavaScript.

## Features
- Player spaceship with smooth movement
- Shooting mechanics
- Enemy waves with increasing difficulty
- Score tracking
- Game over and restart functionality

## How to Play
1. Open `index.html` in a modern web browser
2. Use **LEFT/RIGHT arrow keys** or **A/D** to move
3. Press **SPACEBAR** to shoot
4. Destroy enemies to score points
5. Avoid enemy fire and collisions
